Having Trouble Previewing URLs?

You're on the Right Place

All it needs is One Script and you're Done.

Now your website can preview URLs with this service, saving your time and money.

How it Works?

The script helps to pick the divs with class name 'preview' which ultimately takes out the URL from the div.
After extracting the necessary data for previewing from the URL a div is send back to the corresponding
div with all the previewing material packed in it.

How to Use?

Previewing URLs has never been so Easy

You only need to include the following Script

  <script
          src="http://atiya-rabbi.github.io/linkpre.js">
 </script>

  <script
      src="http://atiya-rabbi.github.io/linkpre.js">
 </script>

Steps to follow-


  • Include the JQuery Script in the head tag as follows: 
    <script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
  • Assign the class name of div as 'preview' which contains URL as follows: 
    <div class="preview"><a href="http://thelady25.com"></a></div>
  • You need to include the below script in your html page before closing the body tag as follows: 
    <script src="http://atiya-rabbi.github.io/linkpre.js"> </script> </body>
  • You can style the div with class name 'preview' accordingly

  • Hurrah! now your website is ready to preview URLs
